Denonciation du monde actuel

Voici les différents articles qui coincide tous sur un seul et même thème, les détresses du monde actuel. Nous utiliserons sur ce thème différents types de contenus. Il y aura un article contre une musique, un livre, un lieu, une série, un regret et enfin un espoir.

Une musique interessante

Dans cet article, vous retrouverez la description des paroles de la musique Satellite de Hakai. Contenu de l’article : Avec ces paroles, l’artiste nous montre que monde est obnubiler par l’argent et la consommation. On peut peut-être parler même de consommation de masse.

Le livre pour l’environnement

Dans cet article, vous retrouverez ma description du livre fait par APPA, son titre est ”La pollution de l’air”. Contenu de l’article : Nous allons voir ce que APPA nous apprend sur la pollution de l’air. Mais vu que le livre date de 1998, nous allons voir les tendances de cette époque et les comparer avec les tendances actuelles.

La Série qui résume tout

Dans cet article, vous retrouverez ma description de la série One Piece créée par Eiichirō Oda. Contenu de l’article : Nous allons voir ce que Oda insinuz dans son oeuvre. Il fait pas mal d’allusions cachés ce qui fait son style intrigant et intéressant de l’oeuvre.

le lieu le plus inégalitaire du monde

Dans cet article, vous retrouverez ma description sur le lieu. Le lieu choisi est l’Afrique du sud. Contenu de l’article : Nous allons voir pourquoi nous avons choisi ce lieu. L’Afrique Du Sud est le pays le plus inégalitaires du monde.

Un grand Espoirs de ce monde

Dans cet article, vous retrouverez ma description sur un espoir de ma génération. Mon espoir c’est les personnes aidant les plus démunies. Contenu de l’article : Nous allons énumérer pas mal de systéme pour l’aide aux personnes démunies. Notamment avec les associations ou même les cagnottes.

Un Regret de cette génération

Dans cet article, vous retrouverez ma description sur un regret de ma génération. Mon regret est sur le thème des inégalités dans le monde alimentairement parlant. Contenu de l’article : Nous allons énumérer pas mal d’inègalité dans le mondes. Avec le mauvais accès à l’eau et à la nourriture dans le monde.
